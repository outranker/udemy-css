There are 4 types of Comninators

1. Adjacent sibling: +

2. Child: >

3. General Sibling: ~

4. Descendant: whitespace


Adjacent

h2 + p {

color: red;

}

Child

h2 > p {

color: red;

}

General Sibling

h2 ~ p {

color: red;

}

Descendant

h2 p {

color: red;

}
